    A stream with an async iterable interface. (PassThrough & AsyncIterable)

    This type enables both stream-based and async iteration access to the same data source.

    Type Parameters

    • TModel

      The type of the object to retrieve.

    Example

    interface User {
        id: string;
        name: string;
    }

    const users: StreamWithAsyncIterable<User> = ...;

    // Method 1
    for await (const user of users)
       console.log(user);

    // Method 2
    users.on('data', (user: User) => console.log(user));
